\doxysubsection{Descripción detallada}
Programa principal para la tokenización de múltiples archivos de texto usando Open\+NLP. 

Este programa permite procesar múltiples archivos de texto proporcionados como argumentos, aplica un modelo de tokenización para separar el texto en tokens, y guarda el resultado en un archivo de salida único.
