Clase para entrenar un modelo de fragmentación gramatical (chunker) utilizando Open\+NLP. 

Este programa utiliza datos de entrenamiento en formato Co\+NLL2000 para generar un modelo de fragmentación, que puede ser usado posteriormente para identificar estructuras gramaticales.
