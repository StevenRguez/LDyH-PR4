\doxysubsection{Descripción detallada}
Clase para entrenar un modelo de etiquetado gramatical (Part-\/of-\/\+Speech) usando Open\+NLP. 

Este programa entrena un modelo de etiquetado gramatical (POS) a partir de un conjunto de datos de entrenamiento. Los modelos generados permiten etiquetar palabras de un texto con su categoría gramatical (por ejemplo, sustantivo, verbo, adjetivo).
