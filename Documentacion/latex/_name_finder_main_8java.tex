\doxysubsection{Descripción detallada}
Programa principal para la detección de nombres propios en texto utilizando Open\+NLP. 

Este programa carga un modelo de detección de entidades nombradas, procesa un conjunto de tokens para identificar nombres propios y muestra los resultados.
